1. Màn hình khởi động


2. Thiết lập mã PIN


  • Vui lòng nhập mã PIN để thiết lập

    3. Nhắc lại mã PIN


  • Vui lòng nhắc lại mã PIN

    4. Màn hình quyền riêng tư


  • Nếu bạn đồng ý với chính sách riêng tư này, nhấn vào nút OK để tiếp tục.Nếu không, nhấn nút Hủy để đóng ứng dụng.
  • Màn hình này chỉ được hiển thị tại lần đầu tiên ứng dụng được mở.

    5. Màn hình chính



  • Ở màn hình này, bạn có thể chọn Âm nhạc, Video hoặc Ảnh để mở xem chi tiết.
  • Bạn còn có thể:
    • Thay đổi mã PIN: Chọn Đổi mật khẩu để mở màn hình thay đổi mật khẩu



    • Mở Hướng dẫn sử dụng: Chọn Hướng dẫn sử dung trên menu. Màn hình hướng dẫn sử dụng sẽ chỉ bạn cách sử dụng toàn bộ chức năng của ứng dụng này



    6. Màn hình Ảnh


  • Ở chế độ bình thường, bạn có thể:
    • Nhấn vào biểu tượng Chế độ màn hình để chuyển đổi giữa cách hiển thị danh sách hoặc kiểu thư viện ảnh.

    • Nhấn vào nút Thêm để mở màn hình Chọn media.

    • Nhấn vào nút Quản lý để chuyển sang chế độ chỉnh sửa

  • Ở chế độ chỉnh sửa, bạn có thể:
    • Hủy ẩn : di chuyển những file đã ẩn về nơi mà nó được lưu trữ trước khi ẩn(người dùng có thể thấy nó).

    • Xóa : xóa những file ẩn (những file được chọn).

    • Hủy : Chuyển về chế độ bình thường.

    • Khôi phục: khôi phục 1 file sao lưu, và nhập tất cả các file ẩn từ file sao lưu vào ứng dụng (file sao lưu phải được tạo bằng ứng dụng này)





    • Sao lưu: tạo 1 file sao lưu vào 1 đường dẫn cụ thể trên thẻ nhớ, file sao lưu đó có thể được phục hồi vào bất cứ thiêt bị Android nào sử dụng ứng dụng này





    • Chọn tất cả : chọn tất cả file ẩn

    • Bỏ chọn tất cả : bỏ chọn tất cả file ẩn

    7. Màn hình Chọn file




  • Ở màn hình này, bạn có thể:

    • Thay đổi chế độ hiển thị giữa Danh sách và Thư viện ảnh



    • Chọn tất cả : chọn tất cả file

    • Bỏ chọn tất cả : bỏ chọn tất cả file

    • Ẩn: ẩn tất cả các file được đọn , và quay về màn hình trước đó(màn hình Âm nhạc,Video hoặc Hình ảnh)

    • Sắp xếp: sắp xếp tất cả file theo tên hoặc theo dung lượng



  • Màn hình Âm nhạc, Video có cùng chức năng với màn hình Ảnh.Chỉ có 1 khác biệt duy nhất: kiểu dữ liệu là video cho màn hình Video, và kiểu dữ liệu nhạc cho màn hình Âm nhạc.
