1. Splash Screen


2. PIN setup


  • Please enter PIN to set up

    3. PIN Confirm


  • Please enter PIN to confirm

    4. Privacy screen


  • If you agree with the privacy, click OK button to continue. Otherwise, click Cancel to close this application.
  • This dialog only show at first time when the app is opened.

    5. Home screen



  • In this screen, you can select Audio, Video, Photo item in order to open Category item.
  • You also can:
    • Reset PIN: select reset PIN item in menu to open reset password dialog



    • Open User Guide: select User Guide item in menu. User Guide dialog will guide you how to use all function of this application



    6. Photo screen


  • In Normal mode, you can:
    • Click Change Mode button to switch between List and gallery.

    • Click Add button to open Picker screen.

    • Click Manage button to change to Edit mode

  • In Edit mode, you can:
    • Unhide : move hidden files (selected files) from this application into its original location

    • Delete : delete hidden files (selected files) from this application.

    • Cancel : cancel Edit mode, turn back to normal mode.

    • Restore: Restore a backup file , and import all hidden files from this backup file into this app(backup file have to be created by this application)





    • Back up: create a backup file into specific location on SDCard, a backup file can be restored into any android device by using this application





    • Select All : select all hidden files

    • Deselect All : deselect all hidden files

    7. Picker screen




  • In this screen, you can:

    • Change mode view between List View and Gallery



    • Select All : select all hidden files

    • Deselect All : deselect all hidden files

    • Hide: hide all selected files and go back Category item screen (Audio,Video or Photo screen)

    • Sort: Sort items by name or by size



  • Audio screen, Video screen have the same function with the Photo screen. There is only one difference: media type is video for Video screen, and audio for Audio screen.
